Open the Credential Manager:

	* Press Windows + S to open the search bar.
	* Type "Credential Manager" and press Enter.
	* In Credential Manager, look for any entries related to Git or GitHub.[Windows Credential]

	  Remove the stored credentials associated with GitHub.

Remove Git Configurations:

	Open the Git Bash terminal.
	Run the following commands to remove the global configurations:

	* git config --global --unset user.name
	* git config --global --unset user.email
	* git config --global --unset credential.helper


Change Global Email Configuration:
	* git config --global user.email "your-new-email@example.com"
	* git config --global user.user "onirstudy"


To view all Git configuration settings, you can use the following command in your terminal or Git Bash:

	* git config --global --list
	* git config --list
	You can also view specific configuration values by specifying the key:
	* git config user.name
	* git config user.email

	OR SET
	* git config user.name onirstudy
	* git config user.email onirstudy@gmail.com
